# phaser-gobang
基于phaser引擎的五子棋游戏（前端代码）

本项目的五子棋是联机版，这是前端代码

使用：*直接git clone或下载zip包* 

项目api.js是服务器地址，目前服务器处于在线状态

[游戏地址](http://cdn.whitedi.com/gobang/)
